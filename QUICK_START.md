## 🐛 Troubleshooting

### Issue: "Module not found" errors
**Solution**: Install required packages
```bash
pip install numpy pandas matplotlib scipy scikit-learn PyQt6
```

### Issue: Colors not showing
**Solution**: Make sure you're running the modified file
```bash
python3 interactive_fitting_gui.py
```

### Issue: Window looks wrong
**Solution**: Check PyQt6 is properly installed
```bash
pip install --upgrade PyQt6
```
