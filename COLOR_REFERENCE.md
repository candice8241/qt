## Color Psychology & Purpose

### Purple Tones (Primary Theme)
- **Light Purple Background** (`#D4CCFF`, `#C7BFFF`, `#E8E4FF`): Calming, creative, sophisticated
- **Purple Borders** (`#B794F6`, `#9575CD`): Defines boundaries, creates visual hierarchy
- **Dark Purple Text** (`#4A148C`, `#7B1FA2`): High contrast for readability, professional

### Pastel Accent Colors
- **Mint Green** (`#A8E6CF`): Positive actions (Save, Apply, Subtract)
- **Light Pink** (`#FFB3BA`): Warning actions (Clear, Reset)
- **Light Blue** (`#A5D8FF`, `#B9DEFF`): Navigation and selection
- **Peach** (`#FFC9A8`): Reset/undo operations
- **Yellow** (`#FFF9DB`, `#FFF4CC`): Information display (results table)

### Functional Colors
- **Cyan** (`#00BCD4`): Coordinates and real-time data
- **Orange** (`#FF6B00`): Section headers and results
- **Red** (`#F44336`): Important controls (Smoothing)
- **Pink** (`#FF4081`): Method and parameter labels

## Design Principles

1. **Contrast**: Dark purple text on light backgrounds ensures readability
2. **Hierarchy**: Each panel has a distinct background color for visual separation
3. **Consistency**: All buttons follow the same border style (2px, rounded corners)
4. **Accessibility**: High contrast ratios for text and backgrounds
5. **Harmony**: Pastel palette creates a cohesive, non-jarring visual experience
6. **Functionality**: Colors indicate button function (green for positive, pink for destructive)

## Customization Guide

To customize colors:

1. **Locate the palette dictionary** in `interactive_fitting_gui.py` (around line 575)
2. **Modify color codes** in the dictionary
3. **Update button backgrounds** in respective `setup_*_panel()` methods
4. **Rerun** the application to see changes

Example customization:
```python
# Change to blue theme
self.palette = {
    'background': '#CCE5FF',      # Light blue
    'primary': '#5C99D6',         # Medium blue
    'text_dark': '#003366',       # Dark blue
    # ... adjust other colors accordingly
}
```

## Testing Colors

To test color combinations:
1. Use online tools like [Coolors](https://coolors.co/) or [Adobe Color](https://color.adobe.com/)
2. Check contrast ratios with [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)
3. Test with different screen brightness and lighting conditions
4. Consider color blindness accessibility with [Color Oracle](https://colororacle.org/)
