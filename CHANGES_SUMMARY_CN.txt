🎯 解决方案

1️⃣  设置右侧边界
   ├─ 设置最大宽度: 1600px
   ├─ 设置最小尺寸: 1200×700
   └─ 效果: 界面不会超出屏幕范围

2️⃣  缩减Load File行尺寸
   ├─ 字体: 10pt → 8pt
   ├─ 按钮高度: 55px → 50px
   ├─ 按钮宽度: 大幅缩减
   │  ├─ Load File → Load (55px)
   │  ├─ Fit Peaks → Fit (50px)
   │  ├─ Save Results → Save (50px)
   │  ├─ Auto Find → Auto (50px)
   │  └─ 其他按钮: 50-60px
   ├─ 间距优化: 4px → 3px
   └─ 总节省: 约200-250px

3️⃣  优化实时位置显示
   ├─ 字体: 9pt → 8pt
   ├─ 最小宽度: 300px → 200px
   ├─ 最大宽度: 新增350px限制
   ├─ 显示格式优化:
   │  ├─ 修改前: "2theta: 12.3456  Intensity: 1234.56"
   │  └─ 修改后: "2θ:12.346 I:1234.6"
   └─ 节省空间: 约43%
